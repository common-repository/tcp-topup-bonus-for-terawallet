== Description ==

Reward your customers with some bonus when they topup their wallet. Give a fixed amount of bonus, or a percentage of the topup amount. Set a minimum topup amount in order to be eligible for the bonus.

== Installation ==

1. Unzip and upload folder to the /wp-content/plugins/ directory.
2. Activate through WordPress plugin dashboard page.
3. Visit TeraWallet > Settings > Topup Bonus.
4. Enable topup bonus & configure how much bonus to reward your customers.
